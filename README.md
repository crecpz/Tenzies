## 玩法
- 用盡可能短的時間讓所有骰子顯示相同數目。
- 點擊骰子來凍結骰子數目。
- 底部的「Roll」按鈕可擲出新的一批骰子。
- 被凍結的骰子數目將不會被更換。
- 保留最佳紀錄至 localStorage（保留通關花費時間最短的一次）。
